# Pixel Blackjack

A pixelated version of the classic Blackjack card game, built for fun and nostalgia.

## Features
- Classic Blackjack rules with a retro pixel art style.
- Single-player and Multi-player modes against the dealer.
- Simple, intuitive controls.

## Installation
1. Clone the repository: `git clone https://github.com/PizzaPixel-Dev/pixel-blackjack.git`
2. Navigate to the project directory: `cd pixel-blackjack`
3. Install dependencies: `npm install` (assuming Node.js project; adjust if different)
4. Run the game: `npm start`

## Usage
- Start a new game and aim for 21 without busting.
- Use mouse to interact.
